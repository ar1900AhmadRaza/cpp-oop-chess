# ♟️ Terminal Chess in C++

## Overview
A fully playable, two-player terminal chess game engineered entirely in C++. This project demonstrates a deep understanding of Object-Oriented Programming (OOP) principles.

## Core Concepts Applied
*   **Encapsulation:** Keeping data secure within classes.
*   **Inheritance & Polymorphism:** Managing different chess pieces dynamically.

## Features
*   Complete validation for all piece movements.
*   Dynamic turn management and state tracking.
*   Check and Checkmate detection algorithms.
